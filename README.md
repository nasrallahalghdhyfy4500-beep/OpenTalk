# منصة مناقشة المشاريع

منصة ويب متكاملة لمناقشة المشاريع والتعاون مع المجتمع.

## المميزات

- ✅ نظام تسجيل وتسجيل دخول آمن
- ✅ إنشاء وإدارة المشاريع
- ✅ مناقشات متعددة لكل مشروع
- ✅ نظام الردود والتعليقات
- ✅ نظام الإعجابات
- ✅ ملفات شخصية للمستخدمين
- ✅ واجهة سهلة وسريعة

## المتطلبات

- PHP 7.4+
- MySQL 5.7+
- خادم ويب (Apache, Nginx, إلخ)

## التثبيت

### 1. تحميل الملفات
انسخ جميع ملفات المشروع إلى مجلد الخادم الخاص بك.

### 2. إنشاء قاعدة البيانات
1. افتح المتصفح وانتقل إلى: `http://localhost/discussion_platform_php/setup_database.php`
2. سيتم إنشاء قاعدة البيانات والجداول تلقائياً
3. احذف ملف `setup_database.php` بعد الانتهاء (لأسباب أمنية)

### 3. تعديل إعدادات الاتصال (اختياري)
إذا كنت تستخدم بيانات اعتماد مختلفة لقاعدة البيانات، عدّل الملف `includes/config.php`:

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASS', '');
define('DB_NAME', 'discussion_platform');
```

### 4. الوصول إلى الموقع
افتح المتصفح وانتقل إلى: `http://localhost/discussion_platform_php/`

## قاعدة البيانات

### الجداول

#### users
- `id` - معرف المستخدم
- `name` - اسم المستخدم
- `email` - البريد الإلكتروني (فريد)
- `password` - كلمة المرور (مشفرة)
- `created_at` - تاريخ الإنشاء
- `updated_at` - تاريخ آخر تحديث

#### projects
- `id` - معرف المشروع
- `title` - عنوان المشروع
- `description` - وصف المشروع
- `owner_id` - معرف صاحب المشروع
- `status` - حالة المشروع (active, archived, completed)
- `created_at` - تاريخ الإنشاء
- `updated_at` - تاريخ آخر تحديث

#### discussions
- `id` - معرف المناقشة
- `project_id` - معرف المشروع
- `user_id` - معرف المستخدم
- `title` - عنوان المناقشة
- `content` - محتوى المناقشة
- `created_at` - تاريخ الإنشاء
- `updated_at` - تاريخ آخر تحديث

#### replies
- `id` - معرف الرد
- `discussion_id` - معرف المناقشة
- `user_id` - معرف المستخدم
- `parent_reply_id` - معرف الرد الأب (للردود المتداخلة)
- `content` - محتوى الرد
- `created_at` - تاريخ الإنشاء
- `updated_at` - تاريخ آخر تحديث

#### likes
- `id` - معرف الإعجاب
- `user_id` - معرف المستخدم
- `discussion_id` - معرف المناقشة (اختياري)
- `reply_id` - معرف الرد (اختياري)
- `created_at` - تاريخ الإنشاء

## الاستخدام

### 1. التسجيل
- انقر على "إنشاء حساب"
- أدخل اسمك وبريدك الإلكتروني وكلمة المرور
- انقر على "إنشاء حساب"

### 2. إنشاء مشروع
- سجل الدخول
- انقر على "إنشاء مشروع جديد"
- أدخل عنوان المشروع والوصف
- انقر على "إنشاء"

### 3. إضافة مناقشة
- انتقل إلى صفحة المشروع
- انقر على "إنشاء مناقشة جديدة"
- أدخل العنوان والمحتوى
- انقر على "إنشاء مناقشة"

### 4. إضافة رد
- انتقل إلى صفحة المناقشة
- أدخل محتوى الرد
- انقر على "إرسال الرد"

### 5. الإعجاب
- انقر على زر الإعجاب (❤️) على أي مناقشة أو رد

## الأمان

- كلمات المرور مشفرة باستخدام bcrypt
- جميع المدخلات معقمة ضد SQL injection
- التحقق من الملكية قبل التعديل أو الحذف
- جلسات آمنة

## الرفع على استضافة مجانية (Infinity Free)

### الخطوات:

1. **إنشاء حساب على Infinity Free**
   - انتقل إلى https://www.infinityfree.net/
   - أنشئ حساباً جديداً

2. **تحميل الملفات**
   - استخدم FTP أو File Manager
   - انسخ جميع ملفات المشروع إلى مجلد `public_html`

3. **إنشاء قاعدة البيانات**
   - استخدم cPanel
   - أنشئ قاعدة بيانات MySQL جديدة
   - عدّل `includes/config.php` بيانات الاتصال الجديدة

4. **تشغيل setup_database.php**
   - افتح `http://yoursite.com/setup_database.php`
   - سيتم إنشاء الجداول تلقائياً
   - احذف الملف بعد الانتهاء

5. **الوصول إلى الموقع**
   - افتح `http://yoursite.com/`
